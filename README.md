# 📊 Sales Performance Analysis Dashboard
An end-to-end Business Intelligence project that transforms raw sales data into valuable business insights using SQL Server and Power BI.
---
# 📌 Project Overview
This project demonstrates the complete Business Intelligence workflow, starting from data preparation and SQL analysis to creating an interactive dashboard in Power BI.
The dashboard provides insights into sales performance, customer behavior, and product profitability to support data-driven decision-making.
---
# 🎯 Project Objectives
- Analyze overall sales performance.
- Identify top-performing customers.
- Evaluate product profitability.
- Monitor discounts and profit margins.
- Support business decision-making using data.

# 📊 Power BI Dashboard
The dashboard consists of three interactive pages.

Features:
- Total Sales
- Net Sales
- Total Profit
- Profit Margin
- Monthly Sales Trend
- Top Products
- Top Customers
---
## 👥 Customer Analysis
Features:
- Total Customers
- Average Sales per Customer
- Average Orders per Customer
- Top Customers by Profit
- Discount Distribution
---
## 📦 Product Analysis

- Category Performance

# 📌 Key Insights

- A small group of customers contributed a significant portion of total profit.
- Higher discounts were associated with lower profit margins.
- Monthly sales showed noticeable seasonal fluctuations.
